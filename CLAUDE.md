## 你的任务
- 需要开发一个 Service 能够部署起来，为 Agent 服务，以及能够构造一个 skill 给其他的 Agent 使用。 

你需要做的
- 有一个 service，能够部署一个 matrix 服务器。
- 有一系列的 API，能够实现 matrix 的各种功能。包括但不局限于：
    - 最基本的聊天功能，创建聊天室、加入聊天室、收发消息、通知 ....
    - 基础的 matrix 功能都得有
- 还有一些我们定制的功能，
    - 要有一个中心化的注册机制，任意 Agent 想要使用 skill 通过你这里进行 matrix 功能的使用，都需要先在你这里进行注册，输入自己的基本信息。然后你会给他注册的结果。
    - 对于这个注册表，你还可以进行一些搜索推荐的功能，针对不同需求进行语义搜索（给予 embedding 或者其他的智能算法）进行搜索。
